PROYECTO 1: CONTADOR POMODORO

Este proyecto consiste basicamente en crear un contador que emplea el método pomodoro. Este tipo de contadores son realmente útiles para estudiantes principalmente, aunque pueden ser igualmente válidos para cualquier acción que requiera concentración mientras llevas un control del tiempo y los descansos.

En este proyecto diseñe una web para un contador Pomodoro, de forma que resultara intuitivo, tanto a la hora de acceder a sus funcionalidades como a la hora de ver la web en pleno funcionamiento.

Cliente X:

Soy profesor de secundaria, necesitaba que los alumnos fueran conscientes del tiempo que toman sus turnos cuando trabajan en grupo. Esta web junto con su diseño tan intuitivo ayudaba realmente a que las clases fueran mucho mas fluidas.


Cliente Y:

A la hora de estudiar en casa, siempre buscaba una web que emplease el método pomodoro que fuera fácil de entender e interpretar. Conocía a Daniel desde hace tiempo y le pedí una web de este estilo que pudiera usar de manera definitiva. Estoy encantado con el resultado.

PROYECTO 2: TIENDA DE ROPA LOCAL

EL proyecto en sí, consiste en un diseño sencillo para una página web local. La idea era que el cliente, o más bien el usuario del sitio web, no tuviese que ir pululando a través de menús sin sentido y muy sobrecargados.

Quisé centrarme en que fuera muy directo con el usuario, y que fuese sobre todo minimalista, con una parrilla simple donde se muestran todos los artículos, y un menú de navegación clásico.

Cliente X:

Minimalista, secillo y limpio. Este proyecto cumple todos los requisitos para clientes como yo que quieren ir directo al grano.


Cliente Y:

Necesitaba un diseño para mi tienda de ropa, contacté con Daniel, y le pedí un diseño sencillo que difiera de las tiendas online de hoy en día, sin menús o secciones muy extravagantes. Aún con su corta trayectoria, ha superado mis expectativas.
